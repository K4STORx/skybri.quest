Plugin List

Plots

Show off your survivial building abilities on the plots for the server to see!
  • /plot claim - Claims plot
  • /plot clear - Erases everything on plot
  • /plot delete - Deletes plot
  • /plot home - Teleports to plot
  • /plot visit - Visit any players plot
  • /plot trust [name] - Allows user to build on plot
  • /plot remove [name] - Removes user from being allowed to build on plot
  • /plot sethome - Sets plot teleport location
  • /plot setbiome [type] - Sets biome of current plot

Wild

Need materials for your plot or Bored of building in a box? Welcome to the Wild!
  • /wild - Teleports you randomly in wild
  • /claim - Claims land around player
  • /trust [name] - Gives another player permission to edit in your claim
  • /untrust [name] - Removes players permission to edit

Teleport

Ability to Teleport to other Players on the Server
  • /tpa [name] - Teleport request to player
  • /tpaccept - Accepts teleport request
  • /tpahere [name] - Teleport request player to you

Homes

Players have three home uses to save private teleport locations around the maps
  • /home [name] - Teleports to named home
  • /sethome [name] - Creates a new home with given name
  • /delhome [name] - Deletes home with given name

Warps

Players have three warps to create public teleport locations around the maps
  • /warp [name] - Teleports to named warp
  • /warp list - Lists public warps
  • /setwarp [name] - Creates a new warp with given name
  • /delwarp [name] - Deletes warp with given name (Must be owner of warp)
